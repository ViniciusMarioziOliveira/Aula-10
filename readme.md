# Calcs - Calculadora Web

## Descrição

O projeto Calcs é uma calculadora web desenvolvida com o objetivo de praticar lógica de programação e manipulação do DOM. A aplicação permite realizar operações matemáticas básicas e avançadas, além de oferecer um sistema de histórico persistente utilizando o armazenamento local do navegador. O sistema facilita a realização de cálculos rápidos com uma interface simples e funcional.

## Funcionalidades

* Realizar operações matemáticas básicas:

  * Soma (+)
  * Subtração (−)
  * Multiplicação (×)
  * Divisão (÷)

* Realizar operações avançadas:

  * Potência (xʸ)
  * Raiz quadrada (√x)

* Sistema de histórico:

  * Armazenamento de cálculos realizados
  * Persistência de dados utilizando LocalStorage
  * Opção de fixar cálculos
  * Opção de limpar histórico

* Validações:

  * Verificação de campos obrigatórios
  * Bloqueio de raiz de números negativos
